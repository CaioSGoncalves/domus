# Domus 🏠

**A lightweight, self-hosted family home app for tasks and scheduling.**

Domus is a minimalist personal command center designed to organize your daily life without the bloat of modern productivity platforms.

---

### Features
* **Tasks:** Simple, fast todo management.
* **Calendar:** Shared family schedules and events.
* **Notifications:** Real-time alerts for what matters.
* **Privacy First:** 100% self-hosted; your data never leaves your home.

### Tech Stack
* **Backend:** Go
* **Frontend:** Templ & HTMX (No heavy JS frameworks)
* **Deployment:** Managed via [Opus](https://github.com/CaioSGoncalves/opus)
